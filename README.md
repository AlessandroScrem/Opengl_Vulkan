# Opengl_Vulkan
Opengl_Vulkan  Engine

## Target scope:
- Engine using  Vulkan or Opengl Graphis API

## Target platform:
- Windows
- Linux
- MacOS

## Prerequisites
- CMake
- Conan.io package manager

## Howto build
```
run conan-setup.bat
cd build
cmake .. cmake .. -G "Visual Studio 16 2019"
cmake --build . --config Release
```
